I took two semester long courses focused on Circuits and Electronics in my sophomore year at Lick-Wilmerding High School. Circuits and Electronics: Analog and Digital is a semester-long Techincal Arts course offered at LWHS (Lick-Wilmerding). In the class, we learned about the basic skills to design and build an analog and digital (programmable) circuit using Arduino. We created an amplifier/speaker and researched how the shape of our speakers and the material we used affects the resonance and sound quality of our speakers. We were able to choose to add certain features that would make the project more difficult- I chose to add a petrometer, two tweeters, and a bluetooth. We also programmed an audio visualizer using a microcontroller that took in an audio input, analysed the audio, and outputed some code that controlled an LED matrix grid panel.

Circuits and Electronics: Device Invention is another semester-long Technical Arts course offered at LWHS. In this class, we continued to develop our skills with designing and building electronic devices and circuits. We learned about data visualization, API, and other topics. For our data visualization project, our group chose to make a data visualization for a chemistry project we had. It was kinda like a science fair project and we had to find a way to display our data. As a result, we ended up making an interactive chart that showed the ideal environment for yeast growht. We were able to have a lot more freedom in this class; we were able to choose what our many of our projects would be. For our internet-connected project, I chose to make a display (in a small acrylic box) with the Arduino screen that displayed the temperature, weather, and date of that day. In addition, there was a button that would switch between screens so that you would also be able to see the MUNI (public transportation) times for my bus stop.

In all our projects from these two classes, we learned the importance of time management, perserverance and prototyping. Through time management, we were able to make sure that our projects would actually get done on time. Perserverance was super important, especially when we had to trouble-shoot our code or look for small mistakes in our circuits.


    This project took most of the semester. Using resistors, transistors, capacitors, inductors, LEDs, speakers, tweeters, and wires, we created a speaker. I also decided to add a bluetooth component so that I would be able to control my speaker remotedly.
    We began our project by looking over schematics and trying out those schematics on a breadboard. After we go it to work on the breadboard, we began soldering on our actual Arduino circuit board. Though I was able to sum up this process in two sentences, the actual process took around 2-3 weeks. After our circuits were finished, we began working on the enclosure. I decided to go with a simple geometric diamond shape. Using the laser cutter, I cut 1/2 inch MDF into the right sizes. Then, I went to the wood shop and used the edge sander to get the right angles so that the pieces would line up. When the enclosure was finished, I put the circuit board into the enclosure and secured the power jacket, potentiometer, speakers, and tweeters.

    For this project, we were given a choice to either create a audio reactive visualizer or a an electronic instrument
    I decided to create an audio reactive visualizer. Using Arduino and a neopixel, I changed the code so that when a certain frequency was played, the LED columns would peak up.

    Code:

    /* Based on Adafruit_NeoMatrix example for single NeoPixel Shield.
    and
    SparkFun Spectrum Shield Demo
    Toni Klopfenstein @ SparkFun Electronics
    February 2015
    https://github.com/sparkfun/Spectrum_Shield
    Modified by A.Kleindolph 10-2015
    */

    #include <Adafruit_GFX.h>
    #include <Adafruit_NeoMatrix.h>
    #include <Adafruit_NeoPixel.h>

    //Declare Spectrum Shield pin connections
    #define STROBE 4
    #define RESET 5
    #define DC_One A0

    //declare matrix shield data pin
    #define PIN 12

    //declare neopixel matrix configuration
    Adafruit_NeoMatrix matrix = Adafruit_NeoMatrix(8, 8, PIN,
            NEO_MATRIX_TOP + NEO_MATRIX_RIGHT +
            NEO_MATRIX_COLUMNS + NEO_MATRIX_PROGRESSIVE,

            NEO_GRB + NEO_KHZ800);

    const uint16_t colors[] = {
            matrix.Color(0,0,128), matrix.Color(0,0,255), matrix.Color(127,255,212), matrix.Color(0,255,127)
    };

    //Define spectrum variables
    int freq_amp;
    int Frequencies_One[7];
    int i;

    void setup() {
            delay(500);
            Serial.begin(57600);

            //matrix setup
            matrix.begin();
            matrix.setBrightness(40);

            //Set spectrum Shield pin configurations
            pinMode(STROBE, OUTPUT);
            pinMode(RESET, OUTPUT);
            pinMode(DC_One, INPUT);
            digitalWrite(STROBE, HIGH);
            digitalWrite(RESET, HIGH);

            //Initialize Spectrum Analyzers
            digitalWrite(STROBE, LOW);
            delay(1);
            digitalWrite(RESET, HIGH);
            delay(1);
            digitalWrite(STROBE, HIGH);
            delay(1);
            digitalWrite(STROBE, LOW);
            delay(1);
            digitalWrite(RESET, LOW);
    }

    /**************************Main Function Loop*****************************/
    void loop() {
            matrix.fillScreen(0); //clear screen
            Read_Frequencies(); //read function
            matrix.show(); //display pixels
    }

    /*******************Pull frequencies from Spectrum Shield********************/
    void Read_Frequencies() {
            //Read frequencies for each band
            digitalWrite(RESET, LOW);


            delay(1);

            for (freq_amp = 0; freq_amp < 7; freq_amp++) //cycle through 7 frequencies
            {
                    Frequencies_One[freq_amp] = ((analogRead(DC_One) + analogRead(DC_One)) / 2); //average two readings
                    digitalWrite(STROBE, HIGH);
                    digitalWrite(STROBE, LOW);
                   
                    Serial.print (Frequencies_One[0]);
                    Serial.print (" ");
                    if (Frequencies_One[1] > 65) {
                            matrix.drawLine(0, 0, 0, Frequencies_One[5] / 50, colors[0]);
                            matrix.drawLine(1, 0, 1, Frequencies_One[5] / 50, colors[0]);
                            matrix.drawLine(2, 0, 2, Frequencies_One[5] / 25, colors[1]);
                            matrix.drawLine(3, 0, 3, Frequencies_One[5] / 25, colors[1]);
                            matrix.drawLine(4, 0, 4, Frequencies_One[3] / 50, colors[2]);
                            matrix.drawLine(5, 0, 5, Frequencies_One[3] / 50, colors[2]);
                            matrix.drawLine(6, 0, 6, Frequencies_One[4] / 25, colors[3]);
                            matrix.drawLine(7, 0, 7, Frequencies_One[4] / 25, colors[3]);
                    }
                    else
                    {matrix.fillScreen(0); //clear screen}
            }

            Serial.println();
            digitalWrite(RESET, HIGH);
            delay(1);
    }

    #include <Adafruit_SSD1306.h>

    #define OLED_RESET D4
    Adafruit_SSD1306 display(OLED_RESET);

    int num = 0;

    void setup() {
            Serial.begin(115200);

            display.begin(SSD1306_SWITCHCAPVCC, 0x3C); // initialize with the I2C addr 0x3C (for the 128x64)
            display.clearDisplay(); // clears the screen and buffer

            display.setCursor(0,0);
            display.setTextColor(WHITE);
            display.setTextSize(1);
            display.println("Hello, world!");
            display.display();

            Particle.subscribe("hook-response/get_weather", gotWeatherData, MY_DEVICES);

            for(int i=0;i <5;i++) {
                    Serial.println("waiting " + String(5-i) + " seconds before we publish");
                    delay(1000);
            }
    }

    void loop() {
            display.clearDisplay(); // clears the screen and buffer
            display.setCursor(0,0);
            Serial.println("Requesting Weather!");
            display.println("Requesting Weather!");
            display.display();
            Particle.publish("get_weather");

            delay(30000);
    }

    void gotWeatherData(const char *name, const char *data) {

            String str = String(data);
            String locationStr = tryExtractString(str, "<location>", "</location>");
            String weatherStr = tryExtractString(str, "<weather>", "</weather>");
            String tempStr = tryExtractString(str, "<temp_f>", "</temp_f>");
            String windStr = tryExtractString(str, "<wind_string>", "</wind_string>");

            if (locationStr != NULL) {
                    Serial.println("At location: " + locationStr);
                    display.println(locationStr);
            }

            if (weatherStr != NULL) {
                    Serial.println("The weather is: " + weatherStr);
                    display.println(weatherStr);
    }

            if (tempStr != NULL) {
                    Serial.println("The temp is: " + tempStr + String(" *F"));
                    display.println(tempStr + String(" *F"));
            }

            if (windStr != NULL) {
                    Serial.println("The wind is: " + windStr);
                    display.println(windStr);
            }

            display.display();
    }

    String tryExtractString(String str, const char* start, const char* end) {
            if (str == NULL) {
                    return "";
            }

            int idx = str.indexOf(start);
            if (idx < 0) {
                    return "";
            }

            int endIdx = str.indexOf(end);
            if (endIdx < 0) {
                    return "";
            }

            return str.substring(idx + strlen(start), endIdx);
    }
