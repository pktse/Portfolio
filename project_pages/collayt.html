Project information

  • Role: Frontend Developer, UI/UX Designer, Graphic Designer
  • Dates: 2020 - 2021
  • Skillset: HTML, CSS, Adobe XD, Adobe Illustrator

Collayt is a site and platform built by UC Berkeley students to help connect students (and alumni) with others through academic opportunities such as side projects and study groups.

My roles included:

  • creating the frontend of our site using HTML and CSS (Frontend developer)
  • designing the UI/UX of the website using Adobe XD while creating and running UI/UX tests, collecting feedback from users, consolidating and presenting results to the team. (UI/UX Design Lead)
  • creating logo, banners, artwork, and error pages using Procreate and Adobe Illustrator. (Graphic Designer)
