Spotify Stats Webapp

Mockup of what I want the home page to look like (prototyped using Figma)

Project information

  • Dates: Spring 2023 - Present
  • Skillset: React, Typescript/Javascript, HTML
(backend Github repo , frontend Github repo)

At the end of every year, Spotify releases a Spotify Wrapped where users are able to see patterns and trends in their listening + see favorite artists and songs. While I love Spotify Wrapped, my only wish was that we didn't have to wait so long to get data about our listening.

I found that Spotify had their own web API that would allow me to access a user's top songs and artists. Using React and Typescript/Javascript, I'm working on creating a webapp that displays the data cleanly.
