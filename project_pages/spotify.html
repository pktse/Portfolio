Pulsify

Home page of Pulsify
Recommendation page that allows users to create a playlist of recommended songs.

Project information

  • Dates: Spring 2023 - Present
  • Skillset: React, RESTful API, Typescript/Javascript, HTML

Pulsify, hosted on Vercel, is a personal React project that utilizes Spotify's WebAPI to display a user's top artists and tracks and creates a playlist of recommended songs based on these top listenings. It's frontend is made using React and Typescript and it's backend makes use of RESTful APIs and Express.js.
