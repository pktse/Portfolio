Regents'​ and Chancellor's Scholars Association (RCSA) is dedicated to helping Regents’ and Chancellor’s Scholarship recipients of UC Berkeley develop and grow, hosting numerous events, like faculty game night and alumni dinners, and providing resources, ranging from mentorship programs to resume assistance. The Regents’ and Chancellor’s Scholars title is awarded to the top 2% of undergraduates at UC Berkeley.

During my time in RCSA, I was a part of the Web Development Committee (webdev). The committee supports and updates the association’s official website . Using numerous tools like HTML/CSS, Javascript, and Python, the committee fixes both front and back-end issues and collaborates with other committees so that the website is up-to-date with the latest information and events. In addition to using these tools, we were able to apply marketable skills like using GIT, practicing code reviews, etc.
